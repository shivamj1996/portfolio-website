HCI E-Commerce Website

HTML/CSS/PHP/JS

For the first part of my Advanced Internet Development module I was set the task to create a website that would demonstrate different PHP techniques that are required when developing advanced websites. For a topic, I chose to create an online shoe retailer.


The first section for my module was Dynamic Web Content. This included displaying data from a database (in my case, products), being able to search and select this data, user registration and login with editable details, server-side security, an admin area with CRUD functionality and OOP Techniques.

The next section was Interface Technologies and frameworks. This involved learning how to incorporate libraries such as AJAX, JQuery and AngularJS into the website in order to make it more dynamic.

The final section was Data Transfer and Web Services. This section had me include API's from Google and Ebay, extracting and encoding data in JSON and XML ,connecting to a web service (weather web service), creating a custom integrated google search, and building a REST web service that allows for dynamic interaction.
