Who am I?

Hi, I'm Shivam Joshi, a third-year Computing student living in Leeds. My focus is primarily front-end, fully responsive web development.

What I do well

I build dynamic websites using HTML, CSS and Javascript.I use these languages to create a front-end that satisfies the user and gives a positive user experience. I'm also fluent in back-end languages such as PHP and MYSQL to display and manage data. Combined, these tools allow me to build fully functional and engaging websites without comprimising on design.

I also have a keen interest in the field of Human Computer Interaction, and during my university course I have learnt how to conduct usability evaluations using a variety of different methods, from creating initial wireframe proptotypes to task-based participant observation and interviews. I can contribute at every step of development and build a website from scratch to completion.

As a web designer I normally use Sass, as well as Gulp when designing projects.
