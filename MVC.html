CodeIgniter MVC Blog Website

HTML/CSS/PHP/CodeIgniter

Part of my Advanced Internet Development module in University involved learning about and implementing an MVC wesbite. I used the Codeigniter framework as it has a lower learning curve compared to other PHP MVC frameworks.

I decided to use the MVC framework to extend my shoe e-commerce website that I made in the first part of the module, and make a blog section for the website. I utilised models, controllers, and partials in order to make a blog system that was connected to my pre-existing MYSQL database. I also included three layers of user profiles (User, Writer, and Moderator), and gave them different user privileges.

Learning about MVC frameworks showed me how much easier topics like database interactivity and form validation could be as opposed to coding scripts from scratch. I hope to continue learning about different MVC frameworks and implement them in future projects.
